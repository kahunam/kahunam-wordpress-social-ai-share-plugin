Enhanced share buttons with integrated AI services for content analysis and sharing optimization.

== Description ==

AI-Enhanced Share Buttons is a powerful WordPress plugin that provides modern share buttons with integrated AI services. It allows visitors to not only share your content on social media but also analyze it with AI tools like ChatGPT, Claude, Gemini, and Perplexity.

= Key Features =

* **AI Service Integration**: Seamlessly integrate with ChatGPT, Claude, Gemini, Perplexity, Grok, and other AI services
* **Smart Prompts**: Hover over AI buttons to reveal dropdown menus with contextual prompts
* **Social Sharing**: Support for all major social networks including Facebook, Twitter/X, LinkedIn, WhatsApp, and more
* **Customizable Design**: Modern pill-style buttons with customizable colors and icons
* **Analytics**: Track button clicks and analyze sharing patterns
* **Flexible Placement**: Auto-display, shortcode, widget, or PHP function placement options
* **Mobile Optimized**: Responsive design with specific mobile layouts
* **Custom Networks**: Add unlimited custom social networks or AI services

= AI Prompt Options =

* Summarize this article
* Extract key stats and bullet points
* Turn into presentation slides
* Fact-check with latest information
* Explain like I'm 5
* Generate discussion questions
* Find related topics

== Installation ==

1. Upload the `ai-share-buttons` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'AI Share Buttons' in the admin menu to configure

== Frequently Asked Questions ==

= How do I add custom AI services? =

Go to AI Share Buttons > Networks in your WordPress admin, click "Add New Network", select "AI Service" as the type, and configure the URL template with appropriate variables.

= Can I customize the prompts? =

Yes! Navigate to AI Share Buttons > AI Prompts to add, edit, or remove prompts and assign them to specific AI services.

= How do I manually place the buttons? =

You can use the shortcode `[ai_share_buttons]`, the PHP function `<?php ai_share_buttons(); ?>`, or the WordPress widget.

= Is click tracking anonymous? =

Yes, the plugin tracks aggregate click data for analytics but does not store personally identifiable information beyond standard web server logs.
